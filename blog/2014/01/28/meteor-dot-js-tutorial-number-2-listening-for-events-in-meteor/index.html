Meteor.JS Tutorial #2: Listening for Events in Meteor

Listening for events in Meteor is super simple, all you need to do is tell meteor which template we're talking about, and what event to listen for.

In your view (.html file)

<body>

</body>

<template name="clickToAlert">
    <button id="alert">Alert Now!</button>
</template>

We have defined a template with a button, and rendered the template using ``. Now let's bind some events.

In your JS file, under the client code, let's write

if (Meteor.isClient) {

    // Here we call Template.TEMPLATE_NAME.events(events_to_bind)
    // Where the `events_to_bind` is a javascript object of events to listen for.
    Template.clickToAlert.events({

        // The structure for the string below is 'event_name jquery_selector'
        /* 
         * Examples:
         * 'dblclick #my-doubleclick-button' would listen for a double click
         * 'keydown input#email' would listed for keydown on the input with ID of "email"
         * you get the idea
        */ 
        'click button#alert' : function() {
            alert('Hey there, you just listened for an event!');
        }
    }
}
