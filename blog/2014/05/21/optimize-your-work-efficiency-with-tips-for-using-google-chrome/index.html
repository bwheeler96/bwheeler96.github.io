Optimize your work efficiency with tips for using Google Chrome

Using Google Chrome is better when you have the inside knowledge. Here are some tips to make Google Chrome better.

Use browser profiles

Browser profiles allow you to switch seamlessly between your digital identities. For me I have a personal profile, a work profile, and separate profiles from different projects I've worked on.

Key points of separate profiles

  • Separate home page(s) for each profile
  • Separate browser history
  • Separate bookmarks

Pinning Tabs

Pinning tabs is great for things you need on the regular, such as your email, Stack Overflow, and your music service. To pin a tab, left click it, and click "Pin tab".

Important hotkeys

  • Cmd-L - Highlights URL Bar
  • Shift-Cmd { - Move one tab left
  • Shift-Cmd } - Move one tab right
  • Cmd ~ - Next window
  • Cmd Click - Open link in new tab
  • Option-Enter - Open search in a new tab
  • Cmd D - Bookmark current tab
  • Shift-Cmd D Bookmark all tabs
  • Cmd F - Find in page

For a full list of hotkeys, check out the google documentation.
